## General info

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).
This project is an app like MS Teams with addition of a GifMaker app. It is written with next.js and TypeScript and deployed on Vercel. It uses MongoDB to save user data and .gif files.

## Installation

- git clone https://github.com/AlaTeam62/GifMaker.git
- npm install

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.js`. The page auto-updates as you edit the file.
